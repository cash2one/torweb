#### flake8

#### Vagrant
类似于VPS，开始时打包开发部署环境

box镜像: http://www.vagrantbox.es/

```
#添加box
vagrant box add boxname ~/box/precise64.box

#实例化一个box
vagrant init boxname

#打包一个配置好的box
vagrant package
```
